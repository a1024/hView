hView 2.0

An image viewer that suports most image formats.
Only the 'v2' folder is needed for compilation.

Dependencies (loaded at runtime):
	FFmpeg   https://ffmpeg.org/download.html
	libraw   https://www.libraw.org/download#stable
	libheif  https://github.com/strukturag/libheif/releases


Building: Run make

Currently Windows-only.
Linux is not yet supported.


Controls: Press F1 for help.
